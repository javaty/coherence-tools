BUILD PREREQUISITES
-------------------

1. Make sure that COHERENCE_HOME environment variable is set properly.
2. Make sure that you have Ant 1.7+ installed and in the path.


BUILD INSTRUCTIONS
------------------

Simply run the following command from the root directory of the project:

> ant
